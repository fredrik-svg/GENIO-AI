Raspberry Pi Voice Assistant — Web UI (Svenska)

- Klicka "Spela in" för att spela in tal (webbläsaren kommer fråga efter mikrofon).
- Inspelningar skickas till servern för transkription via OpenAI Whisper.
- "Säg (TTS)" genererar server-side TTS (espeak-ng) och spelar upp svaret i webbläsaren.

Behöver du koppla in n8n? Starta installationsguiden för n8n-flödet här.
